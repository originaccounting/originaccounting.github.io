无论你是在校学生，还是毕业许久，无论你刚参加工作或者打算换个职位。启源Excel课程都能满足你的需要，从最简单的Ctrl+C入手，到Vlookup，带你全面弥补缺陷，不再让Excel成为你求职路上的短板。

Excel 全面技能课程，4h/节，原价 ，新课程特惠价

Excel基础课程大纲(共2堂课)

Lesson1

1、Excel工作环境认识与设定

  1. Excel工作环境介绍
  2. 自订功能选单
  3. 工作表的新增、删除、移动、複製及隐藏
  4. 显示/隐藏栏或列，插入/删除储存格、栏或列
  5. 调整栏宽和列高，冻结窗格
  6. 自订检视范围，同时检视多工作表

2、Excel基本操作概念

  1. 手动或使用快捷键选取储存格、范围、列或栏
  2. 手动输入资料及编辑储存格内容
  3. 複製和贴上资料，栏位锁定
  4. 相对与绝对位置参照
  5. 跨工作表及跨活页簿参照 ，寻找与取代
  6. 复原、取消复原或重複动作

3、 资料整理

  1. 自动填满与自定义序列填满
  2. 资料排序与自动筛选
  3. 资料验证，依照资料类型进行储存格格式设定
  4. 套用自动表格进行格式设定，条件化进行格式设定
  5. 使用合併彙算整合多工作表内容
  6. 报表製作与格式优化

Session 2

4、 计算

  1. 基本运算子介绍
  2. 资料排序与自动筛选
  3. 常用函数
    • 数学函数
    • 统计函数
    • 查阅与参照函数
    • 资讯函数
    • 逻辑函数
  4. 利用多个函数组合巢状函数

5、 Pivot Table and Pivot Chart

  1. 利用Pivot Table快速撷取报表
  2. 利用Pivot Chart製作动态图表
  3. 图表美化及解读

6、预览及列印工作表

  1. 列印范围及分页设定
  2. 列印固定表头及自订标题和页码
