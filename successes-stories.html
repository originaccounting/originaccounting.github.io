接完大概3次电话，我就不紧张了，把自己想说的都能很好的表达出来，第一次接电话肯定都会紧张所以童鞋们都不用担心，都会词不达意，但是我们都是这么过来的慢慢就好了，而且电话接多了，自己那些问题就都在我们心里了。就拿我这个工作来说吧，她打给我的我没接到，于是我打了回去，接电话的是一家金融公司的hr，已经上班三个月了，我也没具体弄明白我们公司到底是做啥的，简单的讲就是给其他大型公司做公司运营上的战略决策，公司挺大总部在英国，墨村是他们的财务和销售分公司。 好了来和大家分享下我的面试，面试我的一个cfo，一个hr manager，一个ar manager .当时公司招的是个ar officer. 首先AR manager 让我介绍下我在之前公司对于Ar这一块主要有哪些职责。我很详细的讲了，debt collection，credit check, run aging report, bank reconciliation and month end report.我把每一部分都详细说了，也结合了相应的事例进行了阐述。Henry语录：老外普遍喜欢举例说明，会给我们的面试加分。所以在面试前我又请henry老师帮我突击了好几个相关的事例，给力的老师。这部分我大概讲了快30分钟。 接着AR manager 又发问，他问我，平时如何做credit check,我在第一部分介绍我自己的时候已经大概介绍了credit check的部分，我不清楚她是没听清还是我讲的不是她感兴趣的，但既然他又问了，我这次讲的就更具体了。我说我做credit check 通常分两种情况，第一种，如果这个客户是我们的现有客户，那我我会查我们之前的交易记录，如果交易记录良好，那么就正常给他发货。 如果这个客户之前还有一些之前未结的款项，那么我会和他进行沟通，请他把之前的款项结清，我接着给他发这次的货。第二种情况是，如果这个客户是第一次和我们合作的客户， 那么我会请他提供两个以上他们suppliers的电话，进行credit check. 如果口碑良好，我就会正常发货。如果很不幸的，这个客户的口碑不好，那么我会报告给我的manager，及时和她沟通，尽快解决这个问题。Ar manager 又问了，如果付款日期已经过了due date 我通常会怎么做。我说，首先我会发个reminder letter, 如果客户还是没付，那我会发second reminder letter，如果最后还是没反应，那么我会take legal action.接着hr 开始发问，就是说第一部分technical question 已经问完了。首先他让我分析下我自己的优缺点，当然不能说缺点了，缺点就是没有缺点啊哈哈。Henry 老师会帮大家准备的非常完美的。然后又让我说，别人眼中的我，很白痴的问题，我怎么能知道我在别人眼里是啥样啊，但也要表现的很美好，各种憧憬总之哈哈。最后好像问了我是否是个做事在乎细节的人。对的，他们还问了我期待的薪资是多少，我说5万的package.最后我拿到了5万package 的offer.就和大家讲到这吧！

案例六

上周五我通过了三个月的试用期，今天才敢来henry这报道。呵呵。我做的是关于ap的工作，尽管学习会计多年，但真是没有一点经验，毕业后一直在做admin，因为工资比较低增长空间也不大，所以想到了换工作，很幸运通过我学妹的介绍，认识了henry老师，我人生的贵人……

呵呵。因为我白天是full time所以，只能每天晚上下班来上课，真的挺累的那段时期，我是课程系统听完一轮才开始投的简历，我觉得只有把课系统的听一遍才知道，各个职位是做什么的，发展前景怎么样，也可以给自己做一个准确的定位，首先我觉得aa起点比较高，是学会计的人初入职场梦寐以求的职位，但它不适合每个人，例如我，我没有大的梦想，只想稳稳做好一件事，payroll，赚的最多，但也是我认为技术含量最高的职位，接触了才知道，payroll真真儿不是简单发工资啊，呵呵，想想当初多天真啊。所以对于资质平平的我来讲也不适合。而ap ar，我觉得只要认真了，努力了每个人都能找到相关的职位，如果做得好混个manager对于一个女生来讲其实是很不错的。稳稳的，适合我。所以我的简历只准备了，ap ar，而且像我这种想换工作的人来讲真的是没有更多的精力把所有职位都准备的面面俱到，所以以一个过来人的经历来劝奉大家，找工作不要太贪心，只抓自己能抓住的比什么都强。

好，开始聊我的面试吧，我接到电话面试的时候我正在之前的公司上班，所以接电话时候我同事都在我旁边，所以当时电话我听到是interview，我撒了小谎，我说在忙不方便接听，一会打回去。所以我赶忙到公司的会议室给面试的hr回过去。我简单解释了下，刚刚在开会，对方也表示理解。首先他问我为什么要离职，我说因为这份工作是contract,合同要到期了，公司想继续续签，但是我想找个permanent position. 接着他让我讲了下在原公司的major responsibility ,接着问现在公司的工资是多少，我说48000 including super. 最后问我哪天方便过去他们公司进行面试。我面试当天，有cfo, ap manager 和ar manager 一起参与的面试。这家公司是做汽车租赁的，首先ap manager 让我介绍了我每天的工作流程，这部分我已经背了很多遍了，所以讲的很流利也根据了这家公司的情况临场改动了些去迎合他们的需求，这部分我讲了大概30分钟。接着ap manager 又问了我之前公司的 pay run frequency 还有我每周大概要付多少张invoice. 因为之前没有任何经验，所以对于实战的部分，henry老师在课堂上就已将详细介绍过，对于大，中，小型企业每周要处理invoice 的数量以及每个月要付出多少款项，真心觉得henry老师的培训做的太好太到位了，在面试中的任何问题他都提前走在了面试官的前面，让我们这些职场小白可以轻松应对面试中的任何问题。接着ap manager讲，他们每个月最重要的工作就是对账，他问我对账的技巧如何，我又详细的讲了一下对账的流程。他接着问我excel在之前工作中我有用到过吗，我当然说有了，我说我会用excel表格记录我们所有的供货商，和我们付款情况，还有那些供货商在我们提前付款的情况下会给我么相应的discount。力求给工作创造价值最大化。感觉这些面试官听完美美的，哈哈。 这些讲完，又被问道了一些soft question ，这部分我就不详细和大家讲了，因为在henry老师的面试技巧那堂课上涵盖的太全的，所以大家一定要按时听课，因为每堂课必有收获。最后， 他们当场给我下了一个50000的offer, 当我说我要在原公司再做2周因为我要提前给公司notice, 他们也同意了。所以我工作找得很顺利，谢谢henry老师，希望老师培训越做越好，下次跳槽再来找您，哈哈哈！
